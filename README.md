## 🌟 عن المشروع | About

**صِلة** منصة تعليمية متكاملة مصمَّمة خصيصاً لمساعدة الأطفال ذوي الإعاقة السمعية وطيف التوحد على التعلم بأسلوب ممتع وفعّال، مع توفير أدوات متقدمة لأولياء الأمور لمتابعة تقدم أطفالهم.

**Sila** is a comprehensive educational platform designed to help children with hearing impairment and autism spectrum disorder learn in a fun and effective way, while providing advanced tools for parents to track their children's progress.

---

## ✨ المميزات | Features

| الميزة | الوصف |
|--------|-------|
| 🤟 **ترجمة لغة الإشارة** | تحليل وترجمة لغة الإشارة العربية بالذكاء الاصطناعي في الوقت الحقيقي |
| 📚 **دروس تعليمية** | حروف، أرقام، كلمات، جمل، وقصص تفاعلية |
| 🎮 **ألعاب تعليمية** | تعلم بالترفيه عبر ألعاب وكويزات ممتعة |
| 🧩 **اختبارات ABA** | اختبارات مبنية على منهجية ABA العلمية |
| 🌟 **وضع التوحد** | بيئة مصممة خصيصاً بألوان وأنماط مريحة |
| 📊 **لوحة أولياء الأمور** | تقارير تفصيلية عن تقدم الطفل |
| 🧠 **اختبارات CAST** | أدوات الكشف المبكر عن مؤشرات طيف التوحد |
| 🏆 **نظام المكافآت** | نقاط وشارات لتحفيز الأطفال |
